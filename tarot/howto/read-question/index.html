问题解读步骤流程

这里是基于你自己写给自己的问题的塔罗解读的步骤（参考第八课）

  1. 首先准备好

    1. 你的问题
    2. 你的塔罗牌
    3. 你希望使用的牌阵
    4. 卡牌和牌阵的信息页面（如果你需要）

  2. 设置场景和氛围

    1. 按照个人偏好，准备好你的环境
    2. 坐在准备好的空档之前
    3. 放松精神，做几次深呼吸

  3. 思索你的问题

    1. 将卡牌从牌套中取出
    2. 用一只手抓住所有卡牌，另一只手轻轻覆盖在上面
    3. 闭上你的眼睛
    4. 如果你希望的话，可以大声念出你的开场白
    5. 响亮地念出你的问题，如果是背诵的话，注意和你写下问题时的字句保持一致

  4. 洗牌

    1. 洗牌，重复至你认为可以为止
    2. 在洗牌的过程中，保持注意力集中在你的问题上，但也不要过度紧张

  5. 切牌

    1. 将洗好的牌堆竖向放在你面前

    2. 按如下方法随意切牌：

      1. 将牌堆中的一部分拿起来
      2. 把这堆牌放在左边
      3. 然后从新的牌堆中再拿一些牌放在更左边
      4. 随意将这几堆牌放回一起

  6. 布置卡牌

    1. 拿起一张牌（短边向你）
    2. 用你翻开书本的那种样子翻开这种牌
    3. 按照你选择的牌阵把它放到对应位置
    4. 如果你并不准备包括逆转牌，而洗牌过程中又打乱了它，此时将其转回正面

  7. 回应卡牌

    1. 记下你堆每张单卡的感受
    2. 记下你堆整体牌阵的感受

  8. 分析卡牌

    1. 分析每张单卡

      1. 找到那张牌的信息页面
      2. 阅读其中的关键词和动作
      3. 寻找你觉得说的有道理的动作
      4. 写下你的感受和思考，如果你愿意
      5. 考虑卡牌的方向，正向还是逆向？

    2. 分析卡牌之间的关系

    3. 运用解释牌阵的原则

    4. 记下额外的洞察

  9. 创造故事

    1. 大声说出你的故事
    2. 如果你希望的话，可以保留录音

  10. 写下总结

    1. 考虑你的解读过程中的主题或塔罗带给你的消息
    2. 作为总结，用一两句话回答你的问题

  11. 收尾

    1. 记录下卡牌和它们的位置
    2. 清理牌阵
    3. 将卡牌放到一起
    4. 用一只手抓住所有卡牌，另一只手轻轻覆盖在上面
    5. 闭上你的眼睛
    6. 说出你从本次解读中学到的东西
    7. 对你的内心向导表示感谢
    8. 将卡牌放回牌套中
    9. 如果需要的话，恢复你设置的环境

  12. 应用你学到的东西

    1. 基于解读，决定一到两个动作
    2. 记录下你决定的动作
    3. 留意相关事态的后续发展，保持将后续和本次解读联系起来思考
